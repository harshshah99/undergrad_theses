\begin{abstract}
% The Thesis Abstract is written here (and usually kept to just this page).
% The page is kept centered vertically so can expand into the blank space above
% the title too\ldots
Option pricing a very important problem in the field of mathematical finance. Partial differential equation (PDE) based models like Black-Scholes and Heston which are used to price European styles options have been around for a long time. These models provide a satisfactory method to calculate the theoretical value of the premium. With the advent of Deep Learning, models which can learn patterns from data rather than relying on assumptions have started to emerge. 
In this work we explore the effectiveness of neural network based model to price options, specifically the NIFTY 50 Index option between 2017 and 2022.
\end{abstract}
